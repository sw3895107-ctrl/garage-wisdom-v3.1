# Garage Wisdom v3.1

Garage Wisdom is a web-based automotive diagnostic platform that helps users search, decode, and understand vehicle trouble codes (DTCs). The system stores diagnostic information in MongoDB and serves it through a Node.js/Express API with a browser-based interface.

Version 3.1 includes improved deployment support, MongoDB integration, and a simplified frontend.

---

## Features

• Search vehicle diagnostic trouble codes (DTC)  
• MongoDB database storage  
• Node.js + Express backend API  
• Simple web interface  
• Ready for deployment on Render  
• REST API endpoints for future mobile app integration
